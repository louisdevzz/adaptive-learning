## Overview

Adaptive Learning v3.0 is a full-stack application that leverages AI to create personalized learning experiences. The system analyzes student interactions, adapts content difficulty, and provides intelligent recommendations to optimize learning outcomes.

## Tech Stack

### Backend
- ![FastAPI](https://img.shields.io/badge/FastAPI-009688?logo=fastapi&logoColor=white) **FastAPI** - Modern Python web framework for building APIs
- ![SQLAlchemy](https://img.shields.io/badge/SQLAlchemy-D71F00?logo=sqlalchemy&logoColor=white) **SQLAlchemy** - SQL toolkit and ORM
- ![Alembic](https://img.shields.io/badge/Alembic-6BA81E?logo=alembic&logoColor=white) **Alembic** - Database migration tool
- ![PostgreSQL](https://img.shields.io/badge/PostgreSQL-316192?logo=postgresql&logoColor=white) **PostgreSQL** - Relational database for data persistence
- ![Redis](https://img.shields.io/badge/Redis-DC382D?logo=redis&logoColor=white) **Redis** - In-memory data structure store for caching
- ![OpenAI](https://img.shields.io/badge/OpenAI-412991?logo=openai&logoColor=white) **OpenAI API** - Chat + Embeddings for AI-powered features
- ![Pydantic](https://img.shields.io/badge/Pydantic-E92063?logo=pydantic&logoColor=white) **Pydantic v2** - Data validation using Python type annotations
- ![OpenSearch](https://img.shields.io/badge/OpenSearch-005EB8?logo=opensearch&logoColor=white) **OpenSearch** - Search and analytics engine

### Frontend
- ![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=black) **React 18** - UI library with TypeScript
- ![TanStack](https://img.shields.io/badge/TanStack-FF4154?logo=react&logoColor=white) **TanStack Router + Query** - Type-safe routing and data fetching
- ![Rsbuild](https://img.shields.io/badge/Rsbuild-1e40af?logo=rspack&logoColor=white) **Rsbuild (Rspack)** - Fast build tool

### Infrastructure & Deployment
- ![Docker](https://img.shields.io/badge/Docker-2496ED?logo=docker&logoColor=white) **Docker Compose** - Containerization (api, web, db, redis)

### Observability
- ![Sentry](https://img.shields.io/badge/Sentry-362D59?logo=sentry&logoColor=white) **Sentry** - Error tracking (api + web)

### Authentication & Security
- ![JWT](https://img.shields.io/badge/JWT-000000?logo=jsonwebtokens&logoColor=white) **JWT** - Access/Refresh tokens
- **RBAC** - Role-based access control (Admin, Teacher, Student, Parent)

## Key Features

- **Adaptive Learning Paths** - Dynamically adjusts content based on student performance
- **AI-Powered Content** - Uses OpenAI for intelligent content generation
- **Real-time Analytics** - Track student progress and learning patterns
- **Personalized Recommendations** - Suggests next steps based on individual learning style
- **Interactive UI** - Modern, responsive interface built with React
